Mã trạng thái Tên mã Mô tả
200 OK Yêu cầu thành công và server đã trả lời dữ liệu.
201 Created Yêu cầu thành công và tài nguyên đã được tạo mới.
204 No Content Yêu cầu thành công nhưng không có nội dung trả về.
400 Bad Request Yêu cầu không hợp lệ do cú pháp sai hoặc dữ liệu lỗi.
401 Unauthorized Người dùng chưa được xác thực.
403 Forbidden Server hiểu yêu cầu nhưng từ chối thực hiện.
404 Not Found Không tìm thấy tài nguyên yêu cầu.
500 Internal Server Error Server gặp lỗi và không thể hoàn thành yêu cầu.
502 Bad Gateway Server trung gian nhận phản hồi lỗi từ server khác.
503 Service Unavailable Server tạm thời không thể xử lý yêu cầu.
